# c.js

JavaScript -> C transpiler.

## Build

```
$ make
```

### Requirements

Building requires `flex`, `bison`, and `gcc`.

Your *nix distro probably has these bundled.

## Test

```
$ make test
```

### Requirements

Tests require `node`, `npm`, and `ava`.

If you have `npm`, `ava` will be automatically installed in `./node_modules`.
